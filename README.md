## 📝 Kesimpulan Perjalanan Belajar Data Science (Pertemuan 1 - 7)
Selama tujuh pertemuan pertama, saya memperoleh pemahaman yang menyeluruh mengenai fondasi Data Science menggunakan Python. Berikut rangkuman perjalanan belajar saya:
 
- **Pertemuan 1** membuka wawasan tentang ekosistem Data Science secara menyeluruh — mulai dari peran seorang data scientist, siklus hidup proyek data, hingga pengenalan Python dan Jupyter Notebook sebagai lingkungan kerja utama.
- **Pertemuan 2** mempelajari struktur data fundamental di Python seperti list, dictionary, dan tuple, dilanjutkan dengan pengenalan **NumPy** untuk komputasi numerik berbasis array serta **Pandas** untuk pengelolaan data tabular dalam bentuk DataFrame.
- **Pertemuan 3** berfokus pada **Data Cleaning**, yaitu proses membersihkan data mentah agar siap dianalisis. Topik yang dibahas meliputi penanganan *missing values*, deteksi dan penanganan *outlier*, serta teknik ekstraksi informasi dari kolom data yang kompleks.
- **Pertemuan 4** memperkenalkan **Statistika Dasar** sebagai fondasi analisis data, mencakup ukuran pemusatan (mean, median, modus), ukuran penyebaran (variansi, standar deviasi), serta analisis korelasi antar variabel untuk memahami hubungan dalam dataset.
- **Pertemuan 5** mengajarkan teknik **Visualisasi Data** menggunakan **Matplotlib** dan **Seaborn**. Kemampuan menyajikan data secara visual — baik melalui histogram, scatter plot, heatmap, maupun boxplot — sangat penting untuk mengkomunikasikan temuan kepada audiens yang beragam.
- **Pertemuan 6** membahas **Persiapan Data** (*data preparation*) sebelum pemodelan, mencakup encoding variabel kategorikal, normalisasi dan standarisasi fitur, serta teknik pembagian data (*train-test split*). Kualitas data yang disiapkan di tahap ini sangat menentukan performa model.
- **Pertemuan 7** mengantarkan saya ke dunia **Machine Learning** menggunakan scikit-learn, mempelajari konsep dasar supervised learning, membangun model klasifikasi dan regresi sederhana, serta mengevaluasi performa model menggunakan metrik seperti akurasi dan confusion matrix.
Secara keseluruhan, perjalanan belajar ini membentuk pola pikir analitis dan sistematis dalam mengolah data. Setiap pertemuan membangun fondasi untuk topik berikutnya, sehingga proses belajar terasa progresif dan saling berkaitan. Ke depannya, saya berharap dapat mengembangkan kemampuan ini lebih jauh, khususnya pada topik Deep Learning dan analisis data skala besar.
